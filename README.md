# Anil Janjal Portfolio

Professional Next.js website for Anil Janjal. The content includes sections for summary, milestones, skills, experience, projects, education, certification, and contact.

## Structure

- `src/app` - Next.js app router pages and global styles
- `src/components` - Reusable website sections and local SVG icons
- `src/data/resume.js` - Resume content used by the site
- `public` - Static resume PDF and profile image assets

## Scripts

```bash
npm install
npm run dev
npm run build
```
