# Anil Janjal Portfolio

Professional resume portfolio built with Next.js. The content is organized from Anil Janjal's current resume and includes sections for summary, milestones, skills, experience, projects, education, certification, and contact.

## Structure

- `src/app` - Next.js app router pages and global styles
- `src/components` - Reusable portfolio sections and local SVG icons
- `src/data/resume.js` - Resume content used by the site
- `public` - Static resume PDF and profile image assets

## Scripts

```bash
npm install
npm run dev
npm run build
```
